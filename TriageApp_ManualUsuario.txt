# Manual de Usuario - Triage Digital

¡Bienvenido a Triage Digital!

Este manual te guiará para iniciar el sistema por primera vez y comenzar a usarlo en tu computadora con Linux.

---

## 1. Instalación y Primer Inicio

1. **Descomprime o copia la carpeta `TriageApp` en cualquier lugar de tu computadora** (por ejemplo, en tu Escritorio o Documentos).

2. **Abre una terminal y navega hasta la carpeta `TriageApp`:**
   
   ```bash
   cd /ruta/donde/guardaste/TriageApp
   ```

3. **Dale permiso de ejecución al archivo `start` (solo la primera vez):**
   
   ```bash
   chmod +x start
   ```

4. **Inicia el sistema ejecutando:**
   
   ```bash
   ./start
   ```
   
   Esto abrirá el navegador y cargará la pantalla de inicio de sesión de Triage Digital.

---

## 2. Acceso Directo (opcional)

Para mayor comodidad, puedes usar el acceso directo `triage.desktop`:

1. Copia el archivo `triage.desktop` al Escritorio.
2. Haz clic derecho sobre él, selecciona "Permitir ejecutar como programa" o dale permisos con:
   
   ```bash
   chmod +x ~/Escritorio/triage.desktop
   ```
3. Haz doble clic en el icono para iniciar Triage Digital.

---

## 3. Primer Uso del Sistema

- **Usuario y contraseña:**
  - Si es la primera vez que usas el sistema, consulta con el administrador o docente el usuario y contraseña iniciales.
  - Por defecto, suele ser `admin` / `admin` o el que te hayan indicado.

- **Navegación:**
  - Una vez dentro, podrás acceder al panel principal, registrar pacientes, ver listas y utilizar todas las funciones del sistema.

- **Cierre:**
  - Para cerrar el sistema, simplemente cierra la ventana del navegador y la terminal donde se ejecuta Triage.

---

## 4. Requisitos

- Tener Python 3 instalado en la PC (la mayoría de las distribuciones Linux ya lo incluyen).
- No es necesario instalar nada más si usas el ejecutable `start`.

---

¡Listo! Ya puedes comenzar a usar Triage Digital.

Si tienes dudas, consulta este manual o contacta al responsable del sistema.
