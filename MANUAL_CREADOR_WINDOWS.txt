# Manual para crear el ejecutable de Triage en Windows

Este manual es para tu colega encargado de generar el instalador .exe de Triage para Windows.

---

## 1. Requisitos previos
- Tener Windows 10/11
- Instalar Python 3 desde https://www.python.org/downloads/ (marcar "Add Python to PATH")

---

## 2. Instalar dependencias
1. Abrir CMD o PowerShell.
2. Navegar a la carpeta del proyecto Triage.
3. Ejecutar:
   ```cmd
   pip install -r requirements.txt
   pip install pyinstaller
   ```

---

## 3. Usar el script de inicio para Windows
- Usar el archivo `start_windows.py` (ya adaptado para Windows).

---

## 4. Crear el ejecutable
1. En la terminal, ejecutar:
   ```cmd
   pyinstaller --onefile --windowed start_windows.py
   ```
2. Se creará una carpeta `dist/` con el archivo `start_windows.exe`.

---

## 5. Preparar la carpeta para el usuario final
- Crear una carpeta nueva, por ejemplo, `TriageApp`.
- Copiar dentro:
  - `dist/start_windows.exe`
  - La carpeta `triage_digital` (completa)
  - `requirements.txt`
  - El manual de usuario

---

## 6. Probar el ejecutable
- Hacer doble clic en `start_windows.exe` dentro de `TriageApp`.
- El sistema debe abrir el navegador y funcionar sin instalar nada extra.

---

## 7. Notas
- Si hay errores de dependencias, asegúrate de que `requirements.txt` esté actualizado y repite el proceso.
- Si necesitas incluir archivos extra, usa la opción `--add-data` de PyInstaller.

---

¡Listo! Así podrás entregar un ejecutable de Triage para Windows, fácil de usar para cualquier usuario final.
